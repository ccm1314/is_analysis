该类图主要有图书类、预约类、读者类、书目类、借阅记录类、图书管理员类、逾期记录及罚款细则类，其中 书目类与图书类、预约类与图书类、读者类与预约类、借阅记录与读者类存在多对一的关系,书目类与借阅记录类、逾期记录和借阅 记录类存在一对一或者一对零的关系
